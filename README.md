# 픽미업! (Pick Me Up!)

초등 교실 전자칠판용 통합 추첨 포털. 반 명단을 한 번 입력하면 제비뽑기·돌림판(그리고 곧 사다리타기·레이스·캡슐 뽑기)을 골라 실행할 수 있어요. **서버 없음, DB 없음** — 모든 데이터는 브라우저 `localStorage`에만 저장됩니다.

## 실행

```bash
npm install
npm run dev      # 개발 서버
npm run build    # 타입체크 + 프로덕션 빌드
npm run preview  # 빌드 결과 미리보기
```

## 기능

- **Hub**: 반 이름·학생 명단 입력, `이름*3` 가중치, 이번 추첨 제외 체크, 자동 저장(400ms debounce), JSON 내보내기/가져오기, 전부 지우기
- **Lobby**: 상황 필터(1명/여러 명/순서/역할/재미) + 게임 카드 5종
- **제비뽑기**: 한 명 / 여러 명(중복 허용 옵션) / 전원 순서 / 정원 추첨(선발자+대기 순번), 하나씩·한 번에 공개, 쪽지 펼침 Lottie
- **돌림판**: 가중치 반영 칸 크기, 감속 스핀 + 틱 효과음, N명 연속 스핀, 당첨 버스트 Lottie
- **사다리타기**: 랜덤 자동 생성, 구슬 낙하 애니메이션, 라벨 프리셋(당첨/모둠 역할), 전원 공개
- **레이스**: 3-2-1-GO 카운트다운, 이모지 러너, 1등/꼴찌 당첨, 코스 3종, 빨리감기, 실시간 순위
- **캡슐 뽑기**: 캡슐 머신 / 이름 릴 모드, 한 번 더(당첨자 자동 제외)
- **결과 화면**: 당첨자 하이라이트, 선발/대기 결과, 배정 결과, 조 배치·자리 배치, 당첨자 빼고 다시, 결과 복사, 이미지 저장
- 새로고침(F5)·탭 닫기 후에도 명단·설정·마지막 화면 복원, `prefers-reduced-motion` 지원

## 배포

Vercel에 저장소를 import하면 됩니다 (`vercel.json`에 SPA rewrite 포함).
